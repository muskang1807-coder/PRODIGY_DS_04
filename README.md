# Task 4 - Sentiment Analysis on Twitter Data

## Objective

Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes towards specific topics or brands.

---

## Dataset

* **Dataset:** Twitter Training Dataset
* **Source:** Prodigy InfoTech Data Science Internship Dataset

---

## Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Visual Studio Code
* GitHub

---

## Steps Performed

### 1. Data Loading

* Loaded the Twitter dataset into Python using Pandas.

### 2. Data Exploration

* Displayed the first few rows of the dataset.
* Checked dataset information and data types.
* Identified missing values.

### 3. Sentiment Analysis

* Analyzed the distribution of different sentiment categories.
* Counted the number of tweets for each sentiment.

### 4. Data Visualization

Created visualizations to better understand sentiment distribution:

* Bar Chart showing sentiment counts.
* Pie Chart representing sentiment percentages.

## Key Insights

## Conclusion

This project demonstrates the basic workflow of sentiment analysis, including data loading, exploration, analysis, and visualization. It provides insights into how people express different sentiments on social media and helps understand overall public opinion.
